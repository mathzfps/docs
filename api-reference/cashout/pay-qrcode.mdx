---
title: Pagar QR Code
openapi: POST /v1/pix/payment/qrcode
---

Paga um BR Code (copia-e-cola) PIX — dinâmico ou estático.

<Tip>
Recomendamos sempre chamar `/preview` antes pra mostrar os dados ao user e evitar pagamentos errados.
</Tip>

## Fluxo

<Steps>
  <Step title="User cola o BR Code ou escaneia um QR">
    Você obtém uma string do tipo `00020126...`
  </Step>
  <Step title="Chama /v1/pix/payment/qrcode/preview">
    Mostra ao user: valor, nome do recebedor, etc. User confirma.
  </Step>
  <Step title="Chama /v1/pix/payment/qrcode com value_cents confirmado">
    Wallet é debitada. Status `processing`.
  </Step>
  <Step title="Webhook chega com paid ou refunded">
    Em refunded, wallet é restaurada.
  </Step>
</Steps>
