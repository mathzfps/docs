## Como funciona

<Steps>
  <Step title="Pegue o paysure-llm.md">
    Use qualquer uma das 3 ações acima.
  </Step>
  <Step title="Cole no seu agente de I.A.">
    No Claude, ChatGPT, Cursor, Windsurf, GitHub Copilot Chat — qualquer agente que aceite arquivos ou texto longo.
  </Step>
  <Step title="Forneça suas credenciais quando o agente pedir">
    Ele vai precisar do `PAYSURE_CI`, `PAYSURE_CS` e a URL pública onde você quer receber webhooks. Você gera as credenciais no [painel Paysure → API Keys](https://paysure.com.br/).
  </Step>
  <Step title="Pronto">
    O agente implementa: cliente HTTP, geração de QR, pagamento, webhook handler com idempotência, snippets na linguagem do seu projeto. Tudo testável em minutos.
  </Step>
</Steps>
